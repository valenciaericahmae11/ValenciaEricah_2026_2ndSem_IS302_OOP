## Week 1 Exercises

- `hello_world.py`: prints the welcome message
- `variables_demo.py`: demonstrates variables and output
- `student_profile.py`: collects user input and displays the profile
- `calculator.py`: performs basic arithmetic operations

## Week 2 Exercises

- `number_checker.py`: checks whether a number is positive, negative, or zero
- `multiplication_table.py`: prints the multiplication table from 1 to 10
- `login_system.py`: allows up to 3 login attempts
- `even_numbers.py`: prints even numbers from 1 to 50

## Week 3 Exercises

- `greet_function.py`: defines and calls a greeting function
- `calculator_functions.py`: adds two numbers using a function
- `math_module.py`: defines add, subtract, multiply, and divide functions
- `main_program.py`: imports and uses functions from `math_module.py`
- `grade_functions.py`: computes grade average and remark using functions

## Week 4 Exercises

- `list_demo.py`: demonstrates list creation and looping
- `student_list.py`: collects and displays student names in a list
- `dictionary_demo.py`: demonstrates dictionary storage and iteration
- `inventory_system.py`: displays product prices from a dictionary
- `grade_list_analyzer.py`: analyzes a list of grades with average, highest, and lowest

## Week 5 Exercises

- `class_demo.py`: defines a Person class and prints attributes
- `student_class.py`: defines a Student class and displays student data
- `car_class.py`: defines a Car class and displays car details
- `bank_account.py`: defines a BankAccount class with deposit/withdraw methods
- `library_system.py`: creates and displays three book objects

## Machine Problem

- `grade_analyzer.py`: calculates the average of 3 subject grades and shows a remark
