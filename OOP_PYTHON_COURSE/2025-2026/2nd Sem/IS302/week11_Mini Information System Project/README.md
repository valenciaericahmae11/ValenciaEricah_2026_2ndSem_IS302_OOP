# Student Information System

## Description
This project is a simple Student Information System developed using Python.
It demonstrates Object-Oriented Programming, file handling, exception handling,
and a menu-driven system.

## Features
- Add Student
- View Students
- Search Student
- File storage using text files

## Technologies Used
- Python 3

## How to Run the Program

1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the program:

```bash
python main.py
